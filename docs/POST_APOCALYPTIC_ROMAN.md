# Bible narrative post-apocalyptique

## Ton

Radio pirate propre, feu de camp numerique, humour zombie, tendresse basse frequence.

## Phrases produit

- LoRa transporte l'etincelle, le Pi 5 garde le feu.
- Pas de swipe casino, juste des signaux humains.
- Un coeur dans le brouillard, pas un GPS dans la poche.
- Si le zombie lit ce message, il est deja trop long.

## Symboles

| Symbole | Usage |
|---|---|
| `<3` | fallback universel |
| `♡` | signature principale |
| `♥` | mode evenement |
| `☢` | debug/zombie |

## Scene d'ouverture

Le monde n'avait pas vraiment fini. Il avait juste perdu le reseau. Les gens continuaient a chercher du cafe, des nouvelles, parfois une main. Sur le toit d'un immeuble, un petit Pi 5 clignotait comme une bougie têtue. Une radio grattait l'air. Un message arriva : `<3`. Et soudain, le desert eut l'air un peu moins desert.
