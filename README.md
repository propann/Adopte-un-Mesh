# Adopte un Mesh

Rencontre locale, signal faible, intentions fortes.

Ce dépôt démarre le projet Adopt/Adopte un Mesh : une plateforme de rencontre locale autour de Meshtastic, d’un Raspberry Pi 5 et d’une interface web mobile-first.

La vraie fondation arrive sur une branche de bootstrap avec l’API, le site, la documentation Pi 5, le protocole radio et l’étude mini-image de profil.
